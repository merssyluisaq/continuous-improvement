What is the OC Curve?

The Operating Characteristic Curve

For a sampling plan (sample size n, acceptance number c), the OC curve shows the probability of accepting a lot (Pa) against its true % of defectives (p). Its shape reveals how well the plan separates good lots from bad ones — and the two risks it carries: rejecting good lots (α) and accepting bad ones (β).

Real-life use: Before signing an inspection deal, supplier and buyer use the OC curve to check the plan is fair to both. Example: a factory inspects 60 of every 1,000 mufflers and accepts the lot if ≤ 1 is defective — the curve shows how often good lots get returned and bad ones slip through.
The two risks every sampling plan carries
Alpha (α) Risk — Producer / Seller

Rejecting a good lot

The lot meets the AQL (Acceptable Quality Level — the maximum % of defectives the buyer is willing to accept as "good"), but sampling rejects it. The supplier bears the consequences: returns, delays, and loss of trust.

Beta (β) Risk — Consumer / Buyer

Accepting a bad lot

The lot has intolerable quality (LTPD — Lot Tolerance Percent Defective, the % of defectives the buyer considers unacceptable), but sampling approves it. The buyer receives defective products, risking failures and customer complaints.

1. If you increase the sample size n (keeping c the same), what happens to the producer's risk α?
a) It increases — a larger sample finds more defects and rejects more lots
b) It decreases — the curve becomes steeper and better separates good from bad lots
c) It stays the same — n only affects the cost of inspection
Correct! A larger n makes the OC curve steeper — it moves closer to an ideal "step" shape. Both α and β decrease because the sampling plan becomes better at distinguishing good lots from bad ones. Try it in the tool: raise n and watch both risk values drop.
2. The consumer's risk β equals Pa evaluated at which quality level?
a) AQL — the acceptable quality level
b) LTPD — the lot tolerance percent defective
c) The midpoint between AQL and LTPD
Correct! β = Pa(LTPD). Because a lot at the LTPD level is considered intolerable, you want Pa to be very low there — that low value IS β. Ideally β ≤ 10%, meaning the plan rejects at least 90% of lots that bad.
3. You inspect a sample and find d = 3 defectives. The acceptance number is c = 3. What is the decision?
a) Accept the lot
b) Reject the lot
c) Inspect the full lot before deciding
Correct! The rule is: accept if d ≤ c. Here d = 3 and c = 3, so d ≤ c → accept. The lot passes. Try it in the decision box above: set c = 3 and d = 3.
4. What is the only way to completely eliminate both α and β risks?
a) Set a very large acceptance number c
b) Use a very small AQL (close to 0%)
c) Inspect 100% of the lot
Correct! As long as you use sampling, there will always be some probability of a wrong decision — that is statistically unavoidable. Only 100% inspection removes both risks entirely. The OC curve can be made very steep with large n, but it never becomes a perfect step function with finite sampling.
About the calculation

This tool computes the Poisson probability of acceptance directly with the formula Pa = Σ e−λ·λk/k! for k = 0 to c, where λ = n × p. This is the same formula behind the printed Poisson table in your course material. The printed table rounds values and forces you to interpolate between rows, which introduces a tiny rounding error. This tool uses the exact λ value, so results may differ by a few tenths of a percent from the table — both are correct; the table is simply a rounded approximation.

Important note: The α and β risks are not operational errors — they are inherent to every statistical sampling process. The only way to eliminate them completely is through 100% inspection. What we can do is minimize them by adjusting n and c. The recommended producer's risk is ≤ 5% and the consumer's risk ≤ 10%.
